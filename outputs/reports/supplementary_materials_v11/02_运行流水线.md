# v11 论文分析流水线运行说明

## 1. 环境

- Python ≥ 3.10（建议 3.10 或 3.11）
- 操作系统：Windows / Linux / macOS均可
- 从项目根目录 `asd_eeg_aperiodic_study/` 执行下列命令

```powershell
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
pip install mne-icalabel
python scripts/00_check_environment.py
```

## 2. 数据前提

1. 原始 EEG 路径写入 `data/participants/participants.csv`
2. `config/config.yaml` 中路径与采样率、滤波参数与论文方法一致
3. 电极名与 `config/roi_channels.yaml` 中 HydroCel-64（E1–E64）一致

无原始数据时，仍可运行 **07b 之后** 的统计与作图脚本（若仓库中已含 `derivatives/` 与 `outputs/tables/` 中间结果）。

```

```


可选 QC：`19_plot_specparam_qc_review.py`

## 4. 一键运行（Windows）

在项目根目录双击或执行：

```powershell
outputs\reports\supplementary_materials_v11\run_v11_pipeline.bat
```

默认**不**重跑 02–04（预处理）；若需完整重算，请编辑 bat 内 `REM` 注释。

## 5. 与稿件数字的对应

- 主分析 *N* = 138：由 `05_specparam_qc` + `participants` 决定；`08` 读取 QC 后 global 指标
- 表 1 数字：`07b_table1_main_cohort.py` 输出 CSV，与稿中 Table 1 一致时需使用相同 cohort 锁定文件
- ML AUC：以 `25_nested_cv_aperiodic_classifier.py` 运行日志 / `outputs/ml_biomarker/` 为准；随机种子见脚本内 `random_state`

## 6. 常见问题

| 问题 | 处理 |
|------|------|
| 找不到 EEG 文件 | 检查 `participants.csv` 中 `raw_EEG_file` |
| specparam 报错 | `pip install specparam>=2.0.0rc6` |
| ICLabel 未安装 | `pip install mne-icalabel` |
| MixedLM 失败 | 自动降级 OLS，见 `derivatives/logs/roi_mixed_model.log` |
