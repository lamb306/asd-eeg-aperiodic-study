本目录为课程提交用脚本副本（与项目根目录 scripts/ 下文件相同）。

【请勿在本目录直接运行】
副本内 PROJECT_ROOT 仍指向项目根目录的上一级结构；若从本目录调用会导致找不到 config/ 与 src/。

请在项目根目录执行，例如：
  cd d:\asd_eeg_aperiodic_study
  python scripts\08_main_group_analysis.py

或运行：outputs\reports\supplementary_materials_v11\run_v11_pipeline.bat

脚本清单与论文章节对照见上级目录 01_脚本与论文章节对照表.md
